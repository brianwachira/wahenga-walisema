# wahenga-walisema

## Project setup
```
npm install
```
## Install compatible webpack dependency
```
npm install webpack@^4.0.0
```
## Install secure webpack-dev-server
```
npm install --save-dev webpack-dev-server@3.1.14
```
## Add a compatible vue-loader
```
npm add vue-loadeer@14.2.2
```
